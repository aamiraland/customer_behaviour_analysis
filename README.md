# Customer Shopping Behavior Analysis

## Overview

This project is an end-to-end data analytics case study focused on understanding customer shopping behavior. The workflow covers data loading, exploratory data analysis (EDA), data cleaning, SQL-based analysis, dashboard development, and business reporting. The objective is to transform raw customer data into meaningful insights that support data-driven decision making.

---

## Dataset

The dataset contains customer shopping information, including:

* Customer demographics
* Product categories
* Purchase amounts
* Review ratings
* Payment methods
* Discount and promo usage
* Previous purchases
* Shipping type
* Subscription status
* Shopping frequency

## Project Workflow

### 1. Data Loading

* Imported the dataset into Python.
* Performed an initial inspection of data types and missing values.

### 2. Exploratory Data Analysis (EDA)

* Analyzed customer demographics and purchasing patterns.
* Explored distributions, trends, and relationships between variables.
* Identified missing values and inconsistencies.

### 3. Data Cleaning

* Handled missing values.
* Removed duplicates.
* Corrected data types.
* Standardized categorical values.
* Prepared the dataset for analysis.

### 4. SQL Analysis (MySQL)

Imported the cleaned dataset into MySQL and solved business-focused queries, including:

* Top-selling products
* Customer segmentation
* Average purchase amount
* Department and category analysis
* Discount and promotion effectiveness
* Customer purchase frequency
* Product rating analysis
* Revenue-related insights
* Window functions, joins, CTEs, subqueries, and aggregations

### 5. Dashboard Development

Created an interactive Power BI dashboard featuring:

* KPI Cards
* Sales Analysis
* Customer Insights
* Category Performance
* Interactive Filters & Slicers
* Trend Visualizations
* Business Metrics

### 6. Reporting

Prepared a business report and presentation using Gamma to summarize key findings and recommendations.

## How to Run

1. Clone this repository.
2. Open the Jupyter Notebook.
3. Install the required Python libraries.
4. Run the notebook to perform EDA and data cleaning.
5. Import the cleaned dataset into MySQL.
6. Execute the SQL queries.
7. Open the Power BI (.pbix) file to explore the interactive dashboard.
8. Review the project report and presentation for detailed insights.
